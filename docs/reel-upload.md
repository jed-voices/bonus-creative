# Reel upload

The site is wired to expect the demo reel at this path:

`public/reel/2025_bo_wright_demo_reel_v1_1080p.mp4`

## What to do
1. Upload the mp4 file into the `public/reel/` folder in the repository.
2. Keep the filename exactly:
   `2025_bo_wright_demo_reel_v1_1080p.mp4`
3. Once uploaded, the homepage reel button and `/reel/` page will both work.

## Why this is separate
The site code can be committed through the GitHub connector here, but large binary video uploads are more reliable through the GitHub web UI or your local git workflow.
