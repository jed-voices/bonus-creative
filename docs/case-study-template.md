# Case Study Template

## 1. Hero
- Project title
- Category
- Large still frame
- Short one-line summary

## 2. Project overview
Brief description of what the project was and who it was for.

## 3. Client need
What challenge, opportunity, or communication need drove the project?

## 4. Creative approach
How did Bonus Creative think about the story, tone, visual strategy, and structure?

## 5. Production notes
What was handled directly?
Examples:
- concept development
- directing
- shooting
- editing
- drone work
- interview strategy
- event coverage logistics

## 6. Result or impact
What changed because the film was made?
Examples:
- stronger engagement
- clearer messaging
- better donor or audience response
- event momentum
- internal alignment

## 7. Closing CTA
Invite the visitor to start a related project.

Suggested CTA:
If you need a film with a clear point of view and a lean process, start here.
