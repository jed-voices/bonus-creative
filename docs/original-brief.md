# Original Website Brief

The files in this repo were created from the source brief shared in ChatGPT for the Bonus Creative website project.

## Source summary
The original prompt called for:

- a premium, ultra-fast website for an Oklahoma-based solo filmmaker
- a cinematic, editorial, dark, minimal visual direction
- clear positioning around one creator and one vision
- a streamlined sitemap centered on Home, Work, Services, About, Case Studies, and Contact
- fast-loading media, minimal JavaScript, and strong Core Web Vitals
- concise, story-driven copy without fluff or buzzwords
- a site that feels locally credible but nationally competitive

For the full original wording, refer back to the uploaded source brief in the project conversation.
