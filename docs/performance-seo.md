# Performance and SEO Standards

## Performance priorities
- Static-first rendering
- Minimal JavaScript
- Responsive images with exact sizing
- AVIF or WebP wherever possible
- Lazy-load below-the-fold images and media
- Click-to-load video embeds
- Preload only critical assets
- Keep font usage to one display face and one body family
- Use variable fonts when possible
- Keep animations subtle and lightweight

## Hosting recommendations
- Vercel
- Netlify
- Cloudflare Pages

## Framework recommendations
Best fit:
- Astro for maximum performance and low-JS marketing site structure
- Next.js if more complex CMS or future dynamic behavior is expected

## SEO targets
Use locally relevant phrasing naturally, including:
- Oklahoma video producer
- Oklahoma City video production
- branded films
- documentary filmmaker
- nonprofit video production
- church media production
- event video production

## SEO structure
- One clear H1 per page
- Strong page titles and meta descriptions
- Clean heading hierarchy
- Indexable case studies
- Image alt text that is descriptive and natural
- Local business and creative work schema where appropriate

## Metrics to aim for
- Strong mobile Lighthouse scores
- Excellent LCP
- Minimal CLS
- Fast first contentful paint
- Clean accessibility contrast and hierarchy
